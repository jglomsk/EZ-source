We have a couple commands already set up. They are below.

Command = What it does.
<<<<<<<=>>>>>>>>>>>>>>
cmplMac = Compiles the Mac version. NOTE: THIS SERVER IS ON UBUNTU! You cannot run the mac version on the server.
cmplWin = Compiles the Windows version. Run the new file by entering the ~/EZ-source/source directory then typing: wineconsole --backend=curses "ez.exe" "example.ez"
cmplLin = Compiles the Linux version. Run the new file by entering the ~/EZ-source/source directory then typing: ./ez.out example.ez
cmplAll = Compiles all of the versions.
updater = Updates the system that our server is on. Simply grabs updates and installs them.
